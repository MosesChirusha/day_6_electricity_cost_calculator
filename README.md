## Energy Saving Tips

1. **Replace incandescent with LED** — Save up to 80% on lighting
2. **Use timers for water heaters** — Heat only when needed
3. **Set AC to 24-26°C** — Each degree lower uses 6% more power
4. **Unplug devices** — Standby mode still consumes power
5. **Regular maintenance** — Clean AC filters and fridge coils
6. **Use natural ventilation** — Open windows before turning on AC
